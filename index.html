TaiOne Care GPT 是本一科技為台灣居家護理量身打造的 AI 輔助平台，整合硬體 BodyCam、行動 APP 與雲端 AI，重新定義居家護理的工作方式。

一套系統，
串聯護理全流程

從訪視前的 AI 病摘預備、訪視中的即時記錄，到訪視後的資料自動整合，TaiOne Care GPT 讓護理師從繁瑣的文書工作中解放，將精力還給最重要的事——照護病人。

L1

網路層 · 護城河防線

阻絕外部竊聽，全程加密傳輸

  • 所有通訊採用銀行級 HTTPS/TLS 1.2+ 加密
  • VNet 私有網路隔離，應用程式隱藏於公網之外
  • NAT Gateway 固定 IP，統一出站流量路由
L2

認證層 · 三道城門守衛

嚴謹的三層遞進式認證機制

  • Layer 1：API Key 設備驗證，確認合法 App
  • Layer 2：JWT Token 限時通行證，RS256 非對稱加密，1小時自動換鑰
  • Layer 3：身分解析，精準識別機構與角色
L3

授權層 · RBAC 最小權限

機構絕對隔離，杜絕越權存取

  • SuperAdmin → Admin → HeadNurse → Nurse 層級管控
  • OrganizationContext 過濾器，資料邏輯層完全隔離
  • 不同機構使用者在底層 API 即被阻斷
L4

應用層 · 主動防護罩

阻絕惡意行為與異常攻擊

  • Rate Limiting 防止 DDoS 與暴力破解
  • 容量限制 10MB，防止海量資料注入
  • CSRF 保護 + FluentValidation 輸入驗證
L5

資料層 · 數位金庫

最高規格的密碼與機密管理

  • BCrypt 不可逆雜湊，系統中無明文密碼
  • Azure Key Vault 集中管理所有系統鑰匙
  • 首次登入強制重設，Email 驗證啟用帳號
L6

稽核層 · 24小時監控

100% 軌跡追蹤與隱私遮罩

  • 每筆操作記錄來源 IP、裝置、時間、動作
  • Serilog 框架，每日滾動備份，保留 30 天
  • PII 個資自動遮罩，系統管理員亦看不到真實隱私
